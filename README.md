# Spotify_Data_Analysis
Spotify Data Analysis using Python | Exploratory Data Analysis (EDA), Data Visualization, and Music Trend Insights

**Project Overview**
This project analyzes Spotify music data to uncover trends, patterns, and insights related to songs, artists, genres, popularity, and audio features. Using Python and data visualization techniques, the project demonstrates how data analytics can be used to understand listener preferences and music industry trends.


**Objectives**
- Analyze Spotify dataset using Python.
- Perform data cleaning and preprocessing.
- Explore song popularity trends.
- Identify top-performing artists and genres.
- Examine relationships between audio features and popularity.
- Create interactive and insightful visualizations.


**Technologies Used**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook


**Dataset**
The dataset contains Spotify track information including:
- Track Name
- Artist Name
- Genre
- Popularity Score
- Danceability
- Energy
- Loudness
- Acousticness
- Tempo
- Duration


📊 **Analysis Performed**


**Data Cleaning**
- Handled missing values
- Removed duplicates
- Standardized column formats


**Exploratory Data Analysis (EDA)**
- Distribution of song popularity
- Top artists by number of tracks
- Genre analysis
- Correlation analysis of audio features


**Data Visualization**
- Bar Charts
- Histograms
- Heatmaps
- Scatter Plots
- Box Plots


**Key Insights**
- Identified the most popular artists and genres.
- Found correlations between audio features and song popularity.
- Discovered trends in danceability, energy, and listener preferences.
